# Writing Rust (minimal)

- Run `cargo fmt` / `cargo clippy` when appropriate; fix warnings you introduce.
- Prefer `Result` and `thiserror`/`anyhow` patterns already used in the repo.
- See [Rust API guidelines](https://rust-lang.github.io/api-guidelines/) and the official book for depth.
