# Writing TypeScript (minimal)

- Prefer `strict` type settings when the repo enables them.
- Keep `package.json` scripts and tool versions aligned with the lockfile.
- See the [TypeScript handbook](https://www.typescriptlang.org/docs/handbook/intro.html) for language details.
